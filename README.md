## Daily practice ritual

The brain needs time + repetition to weave fluency. Three threads, every day:

- 🔁 **Anki reviews** across 37 themed decks — vocab, grammar, sentence-mining, conjugation, counters, synonyms, register. Pattern + recognition consolidation.
- 📖 **30 minutes of focused reading** per language — recognition reps in authentic context.
- 🎧 **Harry Potter audiobook at 2× speed** — passive sound-recognition training, building the auditory map for all four phonetic systems.
